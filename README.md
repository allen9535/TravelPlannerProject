# 국내 여행 계획 자동 생성 서비스

## 1. 목표와 기능
* 1-1. 목표
    * 일자별 세심한 계획을 통해 더욱 즐거운 경험 도모
    * 국내 여행 계획 수립의 간편화 지향
* 1-2. 기능
    * openAI API를 활용하여 질의·응답
    * 여행지를 국내로 한정하여, 간편한 출발지와 여행지의 선택
